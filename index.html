CryptoTracker

Description:

This web application enables you to enter any cryptocurrency (Bitcoin, Ethereum, etc.) and track it's current value and rate of change, and display the data on a graph.

List of features include:
  • A Live ticker of the current price of Bitcoin in any supported currency
  • A Currency to Bitcoin converter
  • Allowing you to set parameters that will tell you when to buy and sell (based on your input) and populates a graph to track the rate of change after you've "bought in".
My Contributions:
  • Utilized AJAX to create API calls to get cryptocurrency data
  • Utilized JQuery to receive data from user input
  • Combined and debugged the team's code
  • Created the ticker and the currency to Bitcoin converter
  • Utilized Canvas.js to populate and display the data on a graph
Technologies and Frameworks Used:

HTML5, CSS3, JavaScript, JQuery, AJAX, and Canvas.js

Link Inspector

Description:

This Node.js application enables you to enter any Bit.ly link to see the fully resolved URL (where the Bit.ly link leads to), as well as whether or not the link is malicious.

List of features include:
  • Connected to a SQL database to allow instant access to previously scanned links
  • Uses VirusTotal's comprehensive virus scanner to determine if a link is malicious
  • Paired with a Chrome extension that gives you the application's full functionality in your browser by hovering over Bit.ly links.
My Contributions:
  • Utilized Sequelize.js to create SQL queries to store, update, and retrieve links.
  • Using Express.js and Node.js, developed a RESTful API for utlizing CRUD methods.
  • Created server-side code to retrieve the fully resolved URL of a link, and whether or not it's malicious
Technologies and Frameworks Used:

HTML5, CSS3, JavaScript, JQuery, MySQL(Sequlize.js), RESTful Web Services, Node.js, and Express.js
